## Flashcards

At first you need to prepare text file with your questions

```txt
=When II World War started?
==1938
==>1939
==1940
=Which equations equals 4?
==>2+2
==2%2
==>2*2
==>2^2
=Which equations equals 3?
==>2+1
==3%3
==>3*1
==>4-2

```
